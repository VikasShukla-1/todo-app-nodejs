# Permalist App 📝

A full-stack ToDo application built with **Node.js, Express, PostgreSQL, and EJS**.

## Features
- Add new tasks
- Edit tasks
- Delete tasks
- Stores data in PostgreSQL

## Tech Stack
- Node.js
- Express.js
- PostgreSQL
- EJS
- CSS
